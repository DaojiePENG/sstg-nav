Vite Server HTTP API

方法路径功能
POST/api/chat/streamLLM 流式聊天 (SSE)
GET/api/chat/eventsSSE 实时推送
GET/api/chat/sessions会话列表
POST/api/chat/sessions创建会话
PUT/api/chat/sessions/:id重命名会话
DEL/api/chat/sessions/:id删除会话
GET/api/llm-configLLM 配置
PUT/api/llm-config更新 LLM 配置
GET/api/images/*图片静态服务
GET/api/topo/search拓扑图片搜索
POST/api/system/start启动 ROS2 后端
POST/api/system/stop停止 ROS2 后端
